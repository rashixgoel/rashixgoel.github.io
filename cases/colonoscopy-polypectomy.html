Case Study: Colonoscopy with Multiple Polypectomy

Procedure: Colonoscopy with polypectomy
Setting: Surgical Day Care (SDC)

  • Screening colonoscopy vs diagnostic colonoscopy

🧠 Coding Rationale & Decision Points

💡 Critical Skill: Understanding when to code findings vs indication, and how to handle multiple anatomical sites!

Multiple Anatomical Sites = Multiple Diagnosis Codes

The Rule:

When benign neoplasms (polyps) are found at DIFFERENT anatomical sites in the colon, each site gets its own diagnosis code.

In This Case:

Location Code Polyps
Descending colon D12.4 2 polyps (6mm + 4mm)
Sigmoid colon D12.5 1 polyp (8mm)
✅ Why BOTH are Type M:
Each anatomical site required therapeutic intervention (polypectomy). Both are equally "responsible" for the visit - neither is more significant than the other.

What if all 3 polyps were in the SAME location?

If all 3 polyps were found in descending colon only → Code D12.4 once as Type M

Screening Code is ALWAYS Secondary to Findings

The Principle:

When a screening exam FINDS something that requires treatment, the FINDING becomes the main diagnosis, not the screening indication.

Coding Logic:

If colonoscopy found NOTHING:
→ Z12.1 would be Type M (main reason for visit)

If colonoscopy found POLYPS (like this case):
→ D12.x becomes Type M (the actual findings)
→ Z12.1 becomes Type 3 (screening indication)

From Your Textbook (Page 13.18, Example 14):

"Patient brought to day surgery for colonoscopy to investigate diarrhea. No pathology found, final diagnosis: functional diarrhea."

Codes:
M - K59.1 (Functional diarrhea)

💡 The Pattern: Whatever is FOUND (or the reason if nothing found) = Type M. The indication for the test = Type 3.

Multiple Polypectomies = ONE Intervention Code

The CCI Rule:

When multiple polyps are removed during the SAME colonoscopy procedure, code the polypectomy intervention ONCE, regardless of how many polyps were removed.

Why Use Location Attribute "U" (Unqualified)?

Location attribute U = Unqualified or multiple sites within the same organ system

Use "U" when:

  • Polyps removed from multiple sites (descending + sigmoid)
  • Cannot specify a single precise location
  • Intervention affected multiple areas of the same organ

What if Only ONE Polyp in Sigmoid?

If only 1 polyp removed from sigmoid colon → Could use specific location attribute for sigmoid instead of "U"

CCI Code Breakdown:

1.NM.87.BA-U
├─ 1 = Therapeutic intervention
├─ NM = Large intestine (colon)
├─ 87 = Excision, partial
├─ BA = Using colonoscopy
└─ U = Unqualified location (multiple sites)

Inspection Codes Capture the Complete Exam

From Textbook (Page 13.14-13.15):

"A separate code for the endoscopy is required because the endoscopy is extended to a site further than the site where the diagnostic intervention was performed."

Why We Need BOTH Codes:

2.NM.70.BA-BJ (Colonoscopy of large intestine)
Captures the examination of the ENTIRE colon from rectum to cecum
2.NQ.71.BA (Inspection of small intestine)
Captures that the scope went BEYOND the colon into the terminal ileum
The exam was "extended to a site further" than just the colon

What if Terminal Ileum Was NOT Examined?

If scope only examined colon (did not intubate terminal ileum) → Would NOT code 2.NQ.71.BA

💡 Key Concept: The polypectomy code (1.NM.87.BA) shows WHAT was done therapeutically. The inspection codes (2.NM.70, 2.NQ.71) show WHERE and HOW FAR the scope examined.

💡 Interview Scenarios: Practice coding different colonoscopy scenarios!

Question 1: No Polyps Found

Same patient, same screening colonoscopy, BUT no polyps were found. Entire colon and terminal ileum examined - all normal. What changes in coding?

✅ Answer:

Main Diagnosis changes to:
Z12.1 (Type M) - Screening for malignant neoplasm of intestinal tract

Remove: D12.4, D12.5 (no polyps found!)

Interventions stay the same:
- 2.NM.70.BA-BJ (colonoscopy)
- 2.NQ.71.BA (terminal ileum inspection)
- Remove: 1.NM.87.BA (no polypectomy performed)

Rationale: When screening finds nothing abnormal, the screening indication (Z12.1) becomes the main diagnosis.

Question 2: All Polyps in ONE Location

What if all 3 polyps were found in the sigmoid colon only (none in descending colon)? How would diagnosis coding change?

✅ Answer:

Main Diagnosis:
D12.5 (Type M) - Benign neoplasm of sigmoid colon

Remove: D12.4 (no polyps in descending colon)

Keep: Z12.1 (Type 3) - Screening indication

Interventions stay the same - still code polypectomy once

Rationale: Only code anatomical sites where polyps were actually found. Multiple polyps in the SAME site = one diagnosis code for that site.

Question 3: Diagnostic (Not Screening) Colonoscopy

What if this patient came in for colonoscopy because of rectal bleeding (not screening), and the same 3 polyps were found and removed?

✅ Answer:

Main Diagnoses stay the same:
- D12.4 (Type M)
- D12.5 (Type M)

CHANGE:
Remove Z12.1 (this is NOT screening anymore)
Add K62.5 (Type 3) - Hemorrhage of anus and rectum (the symptom that prompted the exam)

Interventions stay the same

Rationale: Z12.1 is ONLY for screening colonoscopies. Diagnostic colonoscopies code the symptom/indication instead (bleeding, diarrhea, abdominal pain, etc.).

Question 4: Biopsy Instead of Polypectomy

What if the physician only BIOPSIED one 3mm polyp in the sigmoid (did not remove it), and the rest of the colon was normal?

✅ Answer:

Main Diagnosis:
D12.5 (Type M) - Benign neoplasm of sigmoid colon (once pathology confirms adenoma)

Keep: Z12.1 (Type 3) if it was screening

Interventions CHANGE:
- Remove: 1.NM.87.BA (polypectomy)
- Add: 2.NM.71.BA (Biopsy, large intestine using colonoscopy)
- Keep: 2.NM.70.BA-BJ (colonoscopy inspection)
- Keep: 2.NQ.71.BA (if terminal ileum examined)

Rationale: Biopsy (taking a small sample) uses a different intervention code than polypectomy (complete removal).
